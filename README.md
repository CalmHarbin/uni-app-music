## 项目的运行

**1. 接口的运行**

```sh
# 下载源码
git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git
# 切换到目录下
cd NeteaseCloudMusicApi
# 安装依赖
cnpm install
# 运行
npm start
```

**2. 项目的运行**

```sh
# 下载源码
git clone https://github.com/CalmHarbin/uni-app-music.git
# 切换到目录下
cd uni-app-music
# 安装依赖
cnpm install
# 运行
npm start
# 使用微信开发者工具打开dist目录即可(请勾上微信开发者工具的 不校验合法域名... 这一项)
```
